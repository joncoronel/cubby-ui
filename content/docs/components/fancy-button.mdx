## API Reference

The FancyButton component is built on top of [Base UI's Button](https://base-ui.com/react/components/button). All Base UI props are supported. The documentation below only covers custom props specific to our implementation.

For the complete Base UI API, see the [Base UI Button documentation](https://base-ui.com/react/components/button).

### Props

#### FancyButton

Stylized button with gradient effects and animations. Wraps Base UI's `Button`.

<ApiPropsList>

<ApiProp
  name="size"
  fullType='"xs" | "sm" | "default" | "lg" | "icon" | "icon_xs" | "icon_sm" | "icon_lg"'
  simpleType="string"
  defaultValue='"default"'
>
  Size of the button. Use icon variants for icon-only buttons with proper square
  dimensions. Custom prop not available in Base UI.
</ApiProp>

<ApiProp name="color" fullType="string" simpleType="string">
  Custom gradient color for the fancy button effect. Custom prop not available
  in Base UI.
</ApiProp>

<ApiProp name="leftSection" fullType="ReactNode" simpleType="node">
  Content (typically icon) to display before the button text. Custom prop not
  available in Base UI.
</ApiProp>

<ApiProp name="rightSection" fullType="ReactNode" simpleType="node">
  Content (typically icon) to display after the button text. Custom prop not
  available in Base UI.
</ApiProp>

</ApiPropsList>
