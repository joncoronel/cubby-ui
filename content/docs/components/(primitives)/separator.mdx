A separator is a thin rule that divides content into groups. Use it between list items, toolbar sections, or stacked cards. Set `orientation="vertical"` to divide items in a row.

## Examples

### Variants

Use `variant` to set the line's thickness and style.

<ComponentPreview component="separator" example="separator-variants" />

| Variant   | Description                          |
| --------- | ----------------------------------- |
| `default` | A 1px solid rule.                   |
| `thick`   | A heavier 2px solid rule.           |
| `dashed`  | A dashed rule for a softer divide.  |

### Vertical

Set `orientation="vertical"` to divide items laid out in a row.

<ComponentPreview component="separator" example="separator-vertical" />

## API Reference

The Separator component is built on top of [Base UI's Separator](https://base-ui.com/react/components/separator). All Base UI props are supported. The documentation below only covers custom props specific to our implementation.

For the complete Base UI API, see the [Base UI Separator documentation](https://base-ui.com/react/components/separator).
