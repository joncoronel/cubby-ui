## API Reference

The Switch component is built on top of [Base UI's Switch](https://base-ui.com/react/components/switch). All Base UI props are supported. The documentation below only covers custom props specific to our implementation.

For the complete Base UI API, see the [Base UI Switch documentation](https://base-ui.com/react/components/switch).

### Props

#### Switch

Toggle control for binary on/off states. Wraps Base UI's `Switch.Root` and `Switch.Thumb`.

<ApiPropsList>

<ApiProp
  name="shape"
  fullType='"circle" | "pill"'
  simpleType="string"
  defaultValue='"circle"'
>
  Controls the aspect ratio and animation travel distance of the switch thumb. Use `"pill"` for a wider, more rectangular thumb.
</ApiProp>

</ApiPropsList>
