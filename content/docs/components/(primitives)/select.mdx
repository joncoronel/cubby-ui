A select lets the user pick one option (or several) from a list bound to a form field. Use it for a known, finite set of choices; for a searchable list of actions or a command palette, use [`Command`](/docs/components/command) instead.

## Composition

```text
Select
├── SelectLabel
├── SelectTrigger
│   └── SelectValue
└── SelectContent
    └── SelectGroup
        ├── SelectGroupLabel
        ├── SelectItem
        └── SelectSeparator
```

`SelectContent` composes the portal, backdrop, positioner, and popup internally. Pass your options as `items` on `Select` and render `SelectItem` children inside `SelectContent`.

### Elevated Variant

On a Card, Dialog, or popover surface, the opaque `default` trigger collapses into its parent. Use `variant="elevated"` on `SelectTrigger` for a translucent overlay that adapts to the substrate. See the [Surfaces](/docs/getting-started/surfaces) docs.

<ComponentPreview component="select" example="select-elevated" />

| Variant    | Description                                                  |
| ---------- | ----------------------------------------------------------- |
| `default`  | Opaque `bg-input` trigger for page-level surfaces.          |
| `elevated` | Translucent trigger for use inside Card, Dialog, or popover. |

## API Reference

The Select component is built on top of [Base UI's Select](https://base-ui.com/react/components/select). All Base UI props are supported. The documentation below only covers custom props and modified defaults specific to our implementation.

For the complete Base UI API, see the [Base UI Select documentation](https://base-ui.com/react/components/select).

### Props

#### SelectLabel

Accessible label for the select trigger. Wraps Base UI's `Select.Label`. Renders a `<div>`: clicking it focuses the trigger without opening the popup.

#### SelectTrigger

Button that opens the dropdown when clicked. Wraps Base UI's `Select.Trigger`.

<ApiPropsList>

<ApiProp
  name="size"
  fullType="'sm' | 'default'"
  simpleType="string"
  defaultValue='"default"'
>
  Size variant of the trigger button.
</ApiProp>

<ApiProp
  name="variant"
  fullType="'default' | 'elevated'"
  simpleType="string"
  defaultValue='"default"'
>
  Surface appearance. `default` uses opaque `bg-input` with a soft drop shadow in light mode. `elevated` uses translucent `bg-input-elevated` for use inside Card, Dialog, or popover surfaces. Matches the same prop on Input, Combobox, and Autocomplete. See the [Surfaces](/docs/getting-started/surfaces) docs.
</ApiProp>

<ApiProp name="hideChevronRotation" fullType="boolean" defaultValue="false">
  Disable the chevron rotation animation when the select opens.
</ApiProp>

</ApiPropsList>

#### SelectContent

Dropdown container for the option list with positioning. Composes Base UI's `Select.Positioner` and `Select.Popup`. Props are forwarded to `Select.Popup`.

<ApiPropsList>

<ApiProp
  name="size"
  fullType="'sm' | 'default'"
  simpleType="string"
>
  Size variant for the popup items. Should match the trigger's size when using
  `alignItemWithTrigger` to ensure proper alignment.
</ApiProp>

<ApiProp name="alignItemWithTrigger" fullType="boolean" defaultValue="false">
  Align the selected item with the trigger button position. Enables native-style
  scrolling with scroll arrows.
</ApiProp>

<ApiProp
  name="fadeEdges"
  fullType="boolean | 'top' | 'bottom' | 'left' | 'right' | 'x' | 'y' | FadeEdge[]"
  defaultValue="true"
>
  Controls the fade effect on scroll edges. See
  [ScrollArea](/docs/components/scroll-area#custom-props) for all options.
</ApiProp>

<ApiProp name="scrollbarGutter" fullType="boolean" defaultValue="false">
  Reserves space for the scrollbar when content overflows, preventing layout
  shift.
</ApiProp>

<ApiProp name="persistScrollbar" fullType="boolean" defaultValue="false">
  Always show the scrollbar instead of fading it in on hover or scroll.
</ApiProp>

<ApiProp name="hideScrollbar" fullType="boolean" defaultValue="false">
  Hides the scrollbar while keeping scroll functionality.
</ApiProp>

<ApiProp name="nativeScroll" fullType="boolean" defaultValue="false">
  Use native browser scrolling with styled scrollbars instead of Base UI's
  scroll virtualization. Enables CSS scroll-driven animations for fade edges.
</ApiProp>

<ApiProp name="sideOffset" fullType="number" defaultValue="6">
  Distance between the trigger and the popup. Base UI defaults to `0`.
</ApiProp>

<ApiProp name="level" fullType="1 | 2 | 3 | 4 | 5 | 6 | 7 | 8" defaultValue="3">
  Surface elevation level for the popup. See the [Surfaces](/docs/getting-started/surfaces) docs. Uses `elevatedSurface` (rim on `::after`) so sticky group labels stay below the rim edge.
</ApiProp>

<ApiProp name="shadowLevel" fullType="1 | 2 | 3 | 4 | 5 | 6 | 7 | 8" defaultValue="3">
  Shadow weight. Controls drop-shadow intensity independently of substrate color.
</ApiProp>

</ApiPropsList>
