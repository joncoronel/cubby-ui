import { Accordion, AccordionContent, AccordionItem, AccordionTrigger } from "@/registry/default/accordion/accordion";

The accordion stacks collapsible sections, each with a header that toggles its content. Use it for FAQs, settings groups, or any list where showing one section at a time keeps the page short.

## Composition

```text
Accordion
└── AccordionItem
    ├── AccordionTrigger
    └── AccordionContent
```

`AccordionTrigger` renders its own `AccordionHeader` internally, so you compose only these four parts.

### Variants

Use `variant` on `Accordion` to change how items are framed and separated.

<ComponentPreview component="accordion" example="accordion-outline-variant" />

| Variant                    | Description                                                       |
| -------------------------- | ---------------------------------------------------------------- |
| `default`                  | Borderless items divided by a bottom rule.                       |
| `split`                    | Each item is its own bordered card.                              |
| `outline`                  | Items grouped inside one bordered card.                          |
| `nested`                   | Indented items inside a muted, rounded container.                |
| `isolated-bordered`        | Bordered items that detach and round when opened.                |
| `isolated-filled`          | Filled items that detach and round when opened.                  |
| `isolated-filled-bordered` | Filled items with dividers that detach and round when opened.    |

## API Reference

The Accordion component is built on top of [Base UI's Accordion](https://base-ui.com/react/components/accordion). All Base UI props are supported. The documentation below only covers custom props and modified defaults specific to our implementation.

For the complete Base UI API, see the [Base UI Accordion documentation](https://base-ui.com/react/components/accordion).

### Props

#### Accordion

Collapsible container that groups expandable content sections. Wraps Base UI's `Accordion.Root`.

<ApiPropsList>

<ApiProp
  name="variant"
  fullType='"default" | "split" | "outline" | "nested" | "isolated-bordered" | "isolated-filled" | "isolated-filled-bordered"'
  simpleType="string"
  defaultValue='"default"'
>
  Visual styling variant for the accordion.
</ApiProp>

<ApiProp
  name="multiple"
  fullType="boolean"
  defaultValue="false"
>
  Allow multiple accordion items to be expanded at the same time. Base UI doesn't set a default.
</ApiProp>

</ApiPropsList>

#### AccordionTrigger

Clickable header that toggles content visibility. Wraps Base UI's `Accordion.Trigger`.

<ApiPropsList>

<ApiProp
  name="showIndicator"
  fullType="boolean"
  defaultValue="true"
>
  Show or hide the expand/collapse indicator icon. Custom prop not available in Base UI.
</ApiProp>

<ApiProp
  name="indicatorType"
  fullType='"chevron" | "plus"'
  simpleType="string"
  defaultValue='"plus"'
>
  Type of indicator icon to display. Custom prop not available in Base UI.
</ApiProp>

<ApiProp
  name="indicatorPosition"
  fullType='"start" | "end"'
  simpleType="string"
  defaultValue='"end"'
>
  Position of the indicator icon relative to the trigger text. Custom prop not available in Base UI.
</ApiProp>

<ApiProp
  name="icon"
  fullType="ReactNode"
>
  Custom icon to display at the start of the trigger. Custom prop not available in Base UI.
</ApiProp>

<ApiProp
  name="subtitle"
  fullType="ReactNode"
>
  Subtitle text to display below the trigger text. Custom prop not available in Base UI.
</ApiProp>

</ApiPropsList>
