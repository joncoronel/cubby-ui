## API Reference

The Popover component is built on top of [Base UI's Popover](https://base-ui.com/react/components/popover). All Base UI props are supported. The documentation below only covers custom props and modified defaults specific to our implementation.

For the complete Base UI API, see the [Base UI Popover documentation](https://base-ui.com/react/components/popover).

### Utilities

#### createPopoverHandle

A function to create a handle for connecting detached triggers to a popover. Useful when the trigger needs to be placed outside the `Popover` component.

```tsx
import { createPopoverHandle } from "@/registry/default/popover/popover";

// Without payload typing
const handle = createPopoverHandle();

// With payload typing
const handle = createPopoverHandle<{ title: string }>();
```

### Props

#### PopoverContent

Composed component for common popover layouts. Combines `Popover.Portal`, `Popover.Positioner`, `Popover.Popup`, and `Popover.Viewport`. Optionally renders `Popover.Arrow`. Props are forwarded to the Positioner and Popup components.

<ApiPropsList>

<ApiProp
  name="arrow"
  fullType="boolean"
  defaultValue="false"
>
  Show or hide the popover arrow pointing to the trigger. Custom prop not available in Base UI.
</ApiProp>

<ApiProp
  name="side"
  fullType="'top' | 'right' | 'bottom' | 'left' | 'inline-start' | 'inline-end'"
  defaultValue='"bottom"'
>
  Which side of the trigger to position the popover. Flattened from Positioner.
</ApiProp>

<ApiProp
  name="align"
  fullType="'start' | 'center' | 'end'"
  defaultValue='"center"'
>
  How to align the popover relative to the trigger. Flattened from Positioner.
</ApiProp>

<ApiProp
  name="sideOffset"
  fullType="number"
  defaultValue="8"
>
  Distance in pixels between trigger and popover. Flattened from Positioner. Base UI defaults to `0`.
</ApiProp>

<ApiProp
  name="alignOffset"
  fullType="number"
  defaultValue="0"
>
  Offset along the alignment axis. Flattened from Positioner.
</ApiProp>

<ApiProp
  name="collisionBoundary"
  fullType="Element | Element[] | 'clipping-ancestors'"
>
  Boundary for collision detection. Flattened from Positioner.
</ApiProp>

<ApiProp
  name="collisionPadding"
  fullType="number | Partial<Record<Side, number>>"
  defaultValue="10"
>
  Padding from collision boundary. Flattened from Positioner. Base UI defaults to `5`.
</ApiProp>

<ApiProp
  name="sticky"
  fullType="boolean"
  defaultValue="false"
>
  Whether popover should stick to trigger during scroll. Flattened from Positioner.
</ApiProp>

<ApiProp
  name="positionMethod"
  fullType="'absolute' | 'fixed'"
  defaultValue='"absolute"'
>
  CSS position method. Flattened from Positioner.
</ApiProp>

<ApiProp
  name="arrowPadding"
  fullType="number"
>
  Minimum padding between arrow and popover edges. Flattened from Positioner. Base UI defaults to `5`.
</ApiProp>

<ApiProp
  name="container"
  fullType="HTMLElement | undefined"
>
  Portal container element. Flattened from Portal.
</ApiProp>

</ApiPropsList>
