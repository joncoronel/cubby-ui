import { Carousel, CarouselContent, CarouselItem, CarouselNext, CarouselPrevious } from "@/registry/default/carousel/carousel";

The carousel cycles through a set of slides one screen at a time, with previous and next controls. Use it for image galleries, feature highlights, or any horizontally or vertically scrollable strip of cards. It is built on Embla Carousel.

## Composition

```text
Carousel
├── CarouselContent
│   └── CarouselItem
├── CarouselPrevious
└── CarouselNext
```

Set `orientation` on `Carousel` to scroll horizontally or vertically.

| Value          | Description                            |
| -------------- | -------------------------------------- |
| `horizontal`   | Slides scroll left and right. Default. |
| `vertical`     | Slides scroll up and down.             |

## Accessibility

`CarouselPrevious` and `CarouselNext` carry visually hidden labels and disable themselves at the ends of the list. The carousel region also handles `ArrowLeft` and `ArrowRight` to move between slides when focused.

## API Reference

The Carousel is built on top of [Embla Carousel](https://www.embla-carousel.com). Slide behavior (loop, alignment, drag, snapping) is configured through the `opts` and `plugins` props, which are forwarded to Embla. The props below are specific to this implementation.

For the full list of Embla options, see the [Embla Carousel options documentation](https://www.embla-carousel.com/api/options/).

### Props

#### Carousel

Root container that creates the Embla instance and shares it through context. Renders a `div` with `role="region"`.

<ApiPropsList>

<ApiProp
  name="orientation"
  fullType='"horizontal" | "vertical"'
  simpleType="string"
  defaultValue='"horizontal"'
>
  Scroll direction. Sets Embla's `axis` to `"x"` or `"y"`.
</ApiProp>

<ApiProp name="opts" fullType="CarouselOptions" simpleType="object">
  Embla options object, forwarded to `useEmblaCarousel` (for example `{ loop: true, align: "start" }`).
</ApiProp>

<ApiProp name="plugins" fullType="CarouselPlugin" simpleType="array">
  Array of Embla plugins, forwarded to `useEmblaCarousel` (for example Autoplay).
</ApiProp>

<ApiProp name="setApi" fullType="(api: CarouselApi) => void" simpleType="function">
  Callback receiving the Embla API instance for imperative control from outside the carousel.
</ApiProp>

</ApiPropsList>

#### CarouselPrevious

Button that scrolls to the previous slide and disables at the start. Wraps [`Button`](/docs/components/button).

<ApiPropsList>

<ApiProp

>
  Button variant. Defaults to `"ghost"` instead of the `Button` default of `"primary"`.
</ApiProp>

<ApiProp

</ApiProp>

</ApiPropsList>
