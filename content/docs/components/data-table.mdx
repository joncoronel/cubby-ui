## Architecture

DataTable uses a **composition-based architecture**. The root `DataTable` component provides context to child components, allowing you to compose the exact table structure you need.

```tsx
<DataTable columns={columns} data={data} enableSorting>
  <DataTableToolbar>
    <DataTableSearch />
    <DataTableColumnToggle />
  </DataTableToolbar>
  <DataTableContent>
    <DataTableHeader enableSorting />
    <DataTableBody emptyState="No results found." />
  </DataTableContent>
</DataTable>
```

## Column Definitions

Define columns using the `ColumnDef` type from TanStack Table. Each column can have:

- `accessorKey` - The key to access data from the row
- `header` - Header content (string or render function)
- `cell` - Cell content render function
- `enableSorting` - Whether the column is sortable (default: `true` when `enableSorting` prop is set)

When `enableSorting` is enabled, columns with string headers automatically render a sortable button with direction indicators. Use `meta.align` to control header alignment, or a render function for fully custom headers.

```tsx
import { type ColumnDef } from "@/registry/default/data-table/data-table"

const columns: ColumnDef<YourDataType>[] = [
  {
    accessorKey: "name",
    header: "Name", // Auto-renders with sort UI when enableSorting is true
  },
  {
    accessorKey: "amount",
    header: "Amount",
    meta: { align: "right" as const }, // Right-align the header
    cell: ({ row }) => {
      const amount = parseFloat(row.getValue("amount"))
      return <div className="text-right">${amount.toFixed(2)}</div>
    },
  },
  {
    accessorKey: "status",
    header: "Status",
    enableSorting: false, // Disable sorting for this column
  },
]
```

## API Reference

### DataTable

Root component that creates the TanStack Table instance and provides context to children.

<ApiPropsList>

<ApiProp
  name="columns"
  fullType="ColumnDef<TData, TValue>[]"
  required
>
  Array of column definitions. See TanStack Table documentation for full options.
</ApiProp>

<ApiProp
  name="data"
  fullType="TData[]"
  required
>
  Array of data objects to display in the table.
</ApiProp>

<ApiProp
  name="children"
  fullType="React.ReactNode"
  required
>
  Composition slot for toolbar, content, and other sub-components.
</ApiProp>

<ApiProp
  name="enableSorting"
  fullType="boolean"
  defaultValue="false"
>
  Enables column sorting at the table level. Pass `enableSorting` to `DataTableHeader` to render sort UI.
</ApiProp>

<ApiProp
  name="enableRowSelection"
  fullType="boolean | ((row: Row<TData>) => boolean)"
  defaultValue="false"
>
  Enables row selection. Pass a function for conditional selection.
</ApiProp>

<ApiProp
  name="enableMultiRowSelection"
  fullType="boolean"
  defaultValue="true"
>
  Allows multiple rows to be selected. Set to `false` for single selection.
</ApiProp>

<ApiProp
  name="showSelectionColumn"
  fullType="boolean"
  defaultValue="true"
>
  When `enableRowSelection` is true, automatically adds a checkbox column. Set to `false` to handle selection UI yourself.
</ApiProp>

<ApiProp
  name="enableFiltering"
  fullType="boolean"
  defaultValue="false"
>
  Enables column filtering.
</ApiProp>

<ApiProp

</ApiProp>

<ApiProp

### DataTableColumnToggle

Dropdown button that allows users to show/hide columns.

This component has no configurable props. It automatically reads column visibility state from the DataTable context.
