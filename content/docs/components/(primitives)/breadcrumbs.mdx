Breadcrumbs show where the current page sits in a site's hierarchy and let users jump back to any ancestor. Use them on deep, nested routes such as docs sections, dashboards, or file paths.

### Variants

## Accessibility

The `Breadcrumb` nav defaults to `aria-label="breadcrumb"`. `BreadcrumbPage` marks the current location with `role="link"`, `aria-disabled="true"`, and `aria-current="page"`, so assistive tech announces it as the current, non-actionable page. `BreadcrumbLink` shows a visible focus ring on keyboard focus. Separators are hidden with `aria-hidden`, and `BreadcrumbEllipsis` carries a screen-reader label ("More pages" by default) for the collapsed items.

### Labeling a collapsed trigger

When you collapse items into a [DropdownMenu](/docs/components/dropdown-menu), give the icon-only trigger an accessible name. `BreadcrumbEllipsis` is hidden from assistive tech, so the button has no name on its own.

```tsx showLineNumbers
<DropdownMenuTrigger render={<Button variant="ghost" size="icon_sm" />}>
  <BreadcrumbEllipsis />
  <span className="sr-only">Toggle menu</span>
</DropdownMenuTrigger>
```

## API Reference

Breadcrumbs are custom styled components built on native HTML elements (`nav`, `ol`, `li`, `a`, `span`). `BreadcrumbLink` is polymorphic through Base UI's `render` prop; the other parts pass through standard attributes for their element.

### Props

#### Breadcrumb

Navigation landmark that wraps the trail. Renders a `nav`.

<ApiPropsList>

<ApiProp name="variant" fullType='"default" | "surface"' simpleType="string" defaultValue='"default"'>
  Visual style of the trail. `default` is a plain inline trail; `surface` frames it in a raised track with the current page lifted onto a pill. Applied through a `data-variant` attribute that descendants read.
</ApiProp>

<ApiProp name="size" fullType='"sm" | "md" | "lg"' simpleType="string" defaultValue='"md"'>
  Scale of the text and spacing for the whole trail, applied through a `data-size` attribute that descendants read.
</ApiProp>

<ApiProp name="aria-label" fullType="string" defaultValue='"breadcrumb"'>
  Accessible label for the navigation landmark.
</ApiProp>

</ApiPropsList>

#### BreadcrumbLink

Interactive ancestor link. Renders an `a` by default.

<ApiPropsList>

<ApiProp name="render" fullType="ReactElement | ((props, state) => ReactElement)" simpleType="element">
  Render the link as a different element, such as a Next.js `Link`. Pass the element and its props are merged in, for example `render={<Link href="/" />}`.
</ApiProp>

</ApiPropsList>

#### BreadcrumbSeparator

Divider between items. Renders an `li` marked `aria-hidden`, defaulting to a chevron icon.

<ApiPropsList>

<ApiProp name="separator" fullType="ReactNode" simpleType="node">
  Custom divider element. Used when no `children` are provided, in place of the default chevron.
</ApiProp>

</ApiPropsList>

#### BreadcrumbEllipsis

Collapsed indicator for the middle of a long path. Renders a `span` marked `aria-hidden` with a screen-reader label.

<ApiPropsList>

<ApiProp name="aria-label" fullType="string" defaultValue='"More pages"'>
  Screen-reader label announced for the collapsed items.
</ApiProp>

</ApiPropsList>
