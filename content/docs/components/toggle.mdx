## API Reference

The Toggle component is built on top of [Base UI's Toggle](https://base-ui.com/react/components/toggle). All Base UI props are supported. The documentation below only covers custom props and modified defaults specific to our implementation.

For the complete Base UI API, see the [Base UI Toggle documentation](https://base-ui.com/react/components/toggle).

### Custom Props

<ApiPropsList>

<ApiProp
  name="variant"
  fullType='"default" | "outline"'
  simpleType="string"
  defaultValue='"default"'
>
  Visual style variant for the toggle button. Custom prop not available in Base UI.
</ApiProp>

<ApiProp
  name="size"
  fullType='"default" | "sm" | "lg"'
  simpleType="string"
  defaultValue='"default"'
>
  Size variant for the toggle button. Custom prop not available in Base UI.
</ApiProp>

</ApiPropsList>
