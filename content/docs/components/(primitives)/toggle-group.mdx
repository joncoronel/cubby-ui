A toggle group joins a set of [`Toggle`](/docs/components/toggle) buttons into one segmented control with shared borders. Use it for related on/off choices that share a slot, like a text alignment or a view switcher, where one or several can be active at once.

## Composition

```text
ToggleGroup
└── Toggle
```

`ToggleGroup` wraps a row of [`Toggle`](/docs/components/toggle) buttons and merges their adjacent borders into one segmented control. Each `Toggle` carries its own `value`.

## API Reference

The ToggleGroup component is built on top of [Base UI's ToggleGroup](https://base-ui.com/react/components/toggle-group). All Base UI props are supported and passed through unchanged. Our implementation only merges the adjacent `Toggle` borders into one segmented control, so there are no custom props or modified defaults.

For the complete Base UI API, including `value`, `defaultValue`, `onValueChange`, `toggleMultiple`, and `disabled`, see the [Base UI ToggleGroup documentation](https://base-ui.com/react/components/toggle-group).
