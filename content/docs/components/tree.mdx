## Examples

### Variants

The Tree component supports three visual variants: `default` (minimal styling), `filled` (card-like with background), and `outline` (bordered card with shadow).

<ComponentPreview component="tree" example="tree-variants" />

### With Icons

Add icons to tree items using the `icon` prop on TreeNode. Use `iconOpen` to show a different icon when the node is expanded (useful for folder states).

<ComponentPreview component="tree" example="tree-with-icons" />

### With Lines

Enable visual connection lines between parent and child nodes using the `showLines` prop for better hierarchy visualization.

<ComponentPreview component="tree" example="tree-with-lines" />

### With Badges

Display additional information using badges. Use the `badge` prop on TreeNode and render with `TreeItemBadge`.

<ComponentPreview component="tree" example="tree-with-badges" />

### With Checkboxes

Enable bulk selection mode with `mode="multiple"`. Parent checkboxes automatically reflect child selection state.

<ComponentPreview component="tree" example="tree-with-checkboxes" />

### Controlled State

Control expansion and selection state externally using the `expanded`/`onExpandedChange` and `selectedNode`/`onNodeSelect` props.

<ComponentPreview component="tree" example="tree-controlled" />

### Disabled & Loading States

Disable interaction on specific nodes using the `disabled` prop. Loading states are shown automatically with a spinner icon when nodes are loading. Set `mode="none"` to create a read-only tree.

<ComponentPreview component="tree" example="tree-disabled-loading" />

### Async Loading

Load tree children dynamically using the `onLoadChildren` callback. The Tree component automatically manages loading state and displays a spinner while fetching data. Children are cached after the first load.

<ComponentPreview component="tree" example="tree-async-loading" />

## API Reference

The Tree component is built with [Base UI's Collapsible](https://base-ui.com/react/components/collapsible) internally for expand/collapse functionality. All Collapsible props are supported on tree nodes. The documentation below only covers custom props specific to our Tree implementation.

For the complete Base UI Collapsible API, see the [Base UI Collapsible documentation](https://base-ui.com/react/components/collapsible).

### Props

#### Tree

Hierarchical view for displaying nested data with selection and expansion. Uses Base UI's `Collapsible` internally.

<ApiPropsList>

<ApiProp name="data" fullType="TreeNode<TData>[]" simpleType="array" required>
  The hierarchical data structure to display.
</ApiProp>

<ApiProp name="children" fullType="(item: TreeNode<TData>) => React.ReactElement" simpleType="function" required>
  Render function for each node.
</ApiProp>

<ApiProp name="selectedNode" fullType="string | null" simpleType="string">
  Selected node ID (controlled).
</ApiProp>

<ApiProp name="onNodeSelect" fullType="(nodeId: string) => void" simpleType="function">
  Callback when node is selected.
</ApiProp>

<ApiProp name="defaultExpanded" fullType="string[]" simpleType="array" defaultValue="[]">
  Initially expanded node IDs (uncontrolled).
</ApiProp>

<ApiProp name="expanded" fullType="string[]" simpleType="array">
  Expanded node IDs (controlled).
</ApiProp>

<ApiProp name="onExpandedChange" fullType="(expanded: string[]) => void" simpleType="function">
  Callback when expanded nodes change.
</ApiProp>

<ApiProp name="checkedNodes" fullType="string[]" simpleType="array">
  Checked node IDs for checkbox mode.
</ApiProp>

<ApiProp name="onCheckedNodesChange" fullType="(checked: string[]) => void" simpleType="function">
  Callback when checked nodes change.
</ApiProp>

<ApiProp name="variant" fullType='"default" | "filled" | "outline"' simpleType="string" defaultValue='"default"'>
  Visual style variant.
</ApiProp>

<ApiProp name="showLines" fullType="boolean" defaultValue="false">
  Show connecting lines between nodes.
</ApiProp>

<ApiProp name="mode" fullType='"single" | "multiple" | "none"' simpleType="string" defaultValue='"single"'>
  Selection mode.
</ApiProp>

</ApiPropsList>

#### TreeNode Interface

TypeScript interface defining the shape of each tree node:
