## API Reference

### ProgressRoot

<ApiPropsList>

<ApiProp name="animated" fullType="boolean" defaultValue="false">
  Enable animated progress transitions for smooth visual updates.
</ApiProp>

<ApiProp
  name="size"
  fullType='"sm" | "md" | "lg"'
  simpleType="string"
  defaultValue='"md"'
>
  Size variant for the progress bar. Choose from small, medium, or large sizes.
</ApiProp>

<ApiProp name="min" fullType="number" defaultValue="0">
  Minimum value of the progress range.
</ApiProp>

<ApiProp name="max" fullType="number" defaultValue="100">
  Maximum value of the progress range.
</ApiProp>

</ApiPropsList>

### ProgressTrack

<ApiPropsList>

<ApiProp name="striped" fullType="boolean">
  Add striped background pattern to the progress track for enhanced visual
  feedback.
</ApiProp>

</ApiPropsList>

### ProgressValue

<ApiPropsList>

<ApiProp
  name="format"
  fullType="(value: number, max: number) => string"
  simpleType="function"
>
  Custom function to format the displayed progress value. Receives current value
  and max value as parameters.
</ApiProp>

</ApiPropsList>

### Additional Props

Progress components inherit all props from [Base UI Progress](https://base-ui.com/react/components/progress#api-reference), including value, defaultValue, onChange, and accessibility props.
