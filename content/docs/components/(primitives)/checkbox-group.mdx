The checkbox group manages the shared state of several related [`Checkbox`](/docs/components/checkbox) controls and wires up a parent checkbox for select-all behavior. Use it for multi-select preference lists, permission sets, or any group where selections combine into one value.

## API Reference

The CheckboxGroup component is built on top of [Base UI's CheckboxGroup](https://base-ui.com/react/components/checkbox-group). All Base UI props are supported and passed through unchanged. Our implementation only adds default layout styling, so there are no custom props or modified defaults.

For the complete Base UI API, including `value`, `defaultValue`, `onValueChange`, and `allValues`, see the [Base UI CheckboxGroup documentation](https://base-ui.com/react/components/checkbox-group).
