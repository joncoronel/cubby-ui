### Elevated Variant

On a Card, Dialog, or popover surface, the opaque `default` group collapses into its parent. Use `variant="elevated"` on `InputGroup` — the inner Input is transparent, so the container's variant controls the entire group's appearance. See the [Surfaces](/docs/getting-started/surfaces) docs.

## API Reference

The Input Group is a custom layout component for combining inputs with addons. All props listed below are custom to this implementation.

### Props

#### InputGroup

Container that combines inputs with addons. Renders a `div` with `role="group"`. Accepts standard `div` props.

<ApiPropsList>

<ApiProp
  name="variant"
  fullType='"default" | "elevated"'
  simpleType="string"
  defaultValue='"default"'
>
  Surface appearance of the group container. `default` uses opaque `bg-input`. `elevated` uses translucent `bg-input-elevated` for use inside Card, Dialog, or popover surfaces. The inner Input/Textarea inherits visually (it's transparent inside the group). See the [Surfaces](/docs/getting-started/surfaces) docs.
</ApiProp>

</ApiPropsList>

#### InputGroupAddon

Container for icons, text, or buttons positioned relative to the input.

<ApiPropsList>

<ApiProp
  name="align"
  fullType='"inline-start" | "inline-end" | "block-start" | "block-end"'
  simpleType="string"
  defaultValue='"inline-start"'
>
  Position of the addon relative to the input. Use `inline-start` for left side, `inline-end` for right side, `block-start` for above, and `block-end` for below.
</ApiProp>

</ApiPropsList>

#### InputGroupButton

Styled button for use inside addons. Wraps the [Button](/docs/components/button) component with size variants optimized for input groups. Accepts all Button props except `size`.

<ApiPropsList>

<ApiProp
  name="variant"
  fullType='"primary" | "neutral" | "secondary" | "outline" | "ghost" | "link" | "destructive"'
  simpleType="string"
  defaultValue='"ghost"'
>
  Visual style variant of the button. Inherits from the Button component.
</ApiProp>

<ApiProp
  name="size"
  fullType='"xs" | "sm" | "icon_xs" | "icon_sm"'
  simpleType="string"
  defaultValue='"xs"'
>
  Size of the button. Use `icon_xs` or `icon_sm` for icon-only buttons.
</ApiProp>

</ApiPropsList>

#### InputGroupText

Styled text container for labels or static content inside addons. Renders a `span`. Accepts standard `span` props.

#### InputGroupInput

Styled input for use inside the group. Wraps the [Input](/docs/components/input) component with transparent styling. Accepts all Input props.

#### InputGroupTextarea

Styled textarea for multi-line inputs. Wraps the [Textarea](/docs/components/textarea) component with transparent styling. Accepts all Textarea props.
