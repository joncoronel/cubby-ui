The slider lets users pick a value, or a range, by dragging a thumb along a track. Use it for volume, brightness, price ranges, and other continuous settings where quick adjustment matters more than typing an exact number.

## Composition

```text
Slider
├── SliderLabel
└── SliderValue
```

`SliderLabel` and `SliderValue` are optional children that render **above** the track, so a header row (the label and value side by side) sits directly above the slider control:

```tsx
<Slider defaultValue={[50]}>
  <div className="flex items-center justify-between">
    <SliderLabel>Volume</SliderLabel>
    <SliderValue />
  </div>
</Slider>
```

For content *below* the track, render it as a sibling of `Slider` rather than a child (for example, a `<FieldDescription>` inside the enclosing `<Field>`).

### Variants

Use `variant` to change the slider's visual style.

| Variant       | Description                                          |
| ------------- | --------------------------------------------------- |
| `default`     | A round thumb centered on the track.                |
| `contained`   | The thumb aligns to the track edges at min and max. |
| `squareThumb` | A square thumb in place of the round default.       |

## Accessibility

The slider is operable from the keyboard: focus a thumb, then use the arrow keys to adjust by `step`, `Home`/`End` to jump to the min and max, and `PageUp`/`PageDown` for larger increments.

Give the slider an accessible name when there is no visible `SliderLabel`. For a range slider, use `getAriaLabel` so each thumb gets its own label.

```tsx showLineNumbers
<Slider
  defaultValue={[20, 60]}
  getAriaLabel={(index) => (index === 0 ? "Minimum price" : "Maximum price")}
/>
```

## API Reference

### Slider

<ApiPropsList>

<ApiProp
  name="variant"
  fullType='"default" | "contained" | "squareThumb"'
  simpleType="string"
  defaultValue='"default"'
>
  Visual style variant for the slider. Choose from default, contained (thumb
  aligned to edges), or squareThumb styles.
</ApiProp>

<ApiProp name="getAriaLabel" fullType="((index: number) => string) | null">
  A function that returns an accessible name for each thumb. Use for sliders without a visible label, or for range sliders where each thumb needs a distinct label.
</ApiProp>

<ApiProp name="showSteps" fullType="boolean" defaultValue="false">
  Show step indicator dots along the slider track for discrete value
  visualization.
</ApiProp>

<ApiProp name="min" fullType="number" defaultValue="0">
  Minimum value of the slider range.
</ApiProp>

<ApiProp name="max" fullType="number" defaultValue="100">
  Maximum value of the slider range.
</ApiProp>

<ApiProp name="thumbAlignment" fullType='"center" | "edge"' simpleType="string">
  Alignment of the slider thumb. Automatically set to 'edge' when variant is
  'contained', otherwise defaults to 'center'.
</ApiProp>

</ApiPropsList>

### Additional Props

Slider inherits all props from [Base UI Slider](https://base-ui.com/react/components/slider#api-reference), including orientation, step, value, defaultValue, onChange, and accessibility props.
