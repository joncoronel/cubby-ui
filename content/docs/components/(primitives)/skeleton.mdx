A skeleton is a shimmering placeholder that holds an element's shape while its content loads. Use it for the initial load of cards, lists, tables, or text, in place of a spinner, to preview the layout that is about to appear.

## API Reference

### Props

#### Skeleton

Animated placeholder that displays a shimmer effect. Can be used standalone with explicit dimensions or as a wrapper around content.

<ApiPropsList>

<ApiProp name="visible" fullType="boolean" defaultValue="true">
  Controls skeleton visibility. When `false` and children are present, renders children directly without the skeleton wrapper.
</ApiProp>

<ApiProp name="animate" fullType="boolean" defaultValue="true">
  Enables the shimmer animation. When `false`, displays a static background.
</ApiProp>

<ApiProp name="inverted" fullType="boolean" defaultValue="false">
  Inverts the shimmer highlight color. By default the sheen is light (white-tinted); when `true` it becomes dark (black-tinted). Use on surfaces where the default highlight is too bright.
</ApiProp>

<ApiProp name="multiline" fullType="boolean" defaultValue="false">
  Renders the skeleton as an inline `<span>` so it can wrap across multiple lines of text. Uses `box-decoration-break: clone` so each line fragment carries the skeleton styling, and `color: transparent` so the underlying text reserves its natural dimensions while staying invisible.
</ApiProp>

<ApiProp name="children" fullType="ReactNode" simpleType="node">
  Content to wrap. When provided, children are rendered but hidden (`invisible`) while the skeleton overlay is visible. The skeleton inherits the children's dimensions.
</ApiProp>

</ApiPropsList>
