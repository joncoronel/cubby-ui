## API Reference

### Props

#### Table

Root container that wraps the table and provides variant context to children via data attributes. Features a card-like appearance with a muted background border.

<ApiPropsList>

<ApiProp
  name="bordered"
  fullType="boolean"
  defaultValue="false"
>
  Adds borders to all table cells for a grid appearance.
</ApiProp>

<ApiProp
  name="striped"
  fullType="boolean"
  defaultValue="false"
>
  Adds alternating background colors to rows for better readability.
</ApiProp>

<ApiProp
  name="hoverable"
  fullType="boolean"
  defaultValue="true"
>
  Enables hover effect on rows.
</ApiProp>

<ApiProp
  name="fadeEdges"
  fullType="boolean | 'x' | 'y' | 'top' | 'bottom' | 'left' | 'right'"
  defaultValue="'bottom'"
>
  Adds gradient fade effect at scroll edges to indicate more content.
</ApiProp>

<ApiProp
  name="nativeScroll"
  fullType="boolean"
  defaultValue="false"
>
  Uses native browser scrollbars instead of custom styled scrollbars.
</ApiProp>

<ApiProp
  name="hideScrollbar"
  fullType="boolean"
  defaultValue="false"
>
  Hides the scrollbar while maintaining scroll functionality.
</ApiProp>

<ApiProp
  name="persistScrollbar"
  fullType="boolean"
  defaultValue="false"
>
  Always shows the scrollbar instead of auto-hiding.
</ApiProp>

<ApiProp
  name="scrollbarGutter"
  fullType="boolean"
  defaultValue="false"
>
  Reserves space for scrollbar to prevent layout shift.
</ApiProp>

</ApiPropsList>

#### TableRow

Represents a row in the table. Supports polymorphic rendering via the `render` prop.

<ApiPropsList>

<ApiProp
  name="selected"
  fullType="boolean"
  defaultValue="false"
>
  Marks the row as selected with a highlighted background.
</ApiProp>

<ApiProp
  name="render"
  fullType="ReactElement | ((props, state) => ReactElement)"
>
  Polymorphic render prop to change the underlying element or customize rendering.
</ApiProp>

</ApiPropsList>
