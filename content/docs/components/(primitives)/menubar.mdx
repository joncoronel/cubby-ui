A horizontal bar of top-level menus, like the File / Edit / View bar in a desktop app. Use it for application-style command surfaces where several menus sit side by side and moving between them keeps the next menu open. For a single menu opened from one button, use [Dropdown Menu](/docs/components/dropdown-menu) instead.

        └── MenubarSub

## API Reference

The Menubar component is built on top of [Base UI's Menubar](https://base-ui.com/react/components/menubar). All Base UI props are supported. The documentation below only covers custom props and modified defaults specific to our implementation.

For the complete Base UI API, see the [Base UI Menubar documentation](https://base-ui.com/react/components/menubar).

### Custom Props

#### Menubar

The bar element itself is a subtle inline toolbar that sits visibly elevated above the page.

<ApiPropsList>

<ApiProp name="level" fullType="1 | 2 | 3 | 4 | 5 | 6 | 7 | 8" defaultValue="2">
  Surface elevation level for the bar. Defaults to `2`, a subtle toolbar that lifts gently from the page. See the [Surfaces](/docs/getting-started/surfaces) docs.
</ApiProp>

<ApiProp name="shadowLevel" fullType="1 | 2 | 3 | 4 | 5 | 6 | 7 | 8" defaultValue="2">
  Shadow weight for the bar.
</ApiProp>

</ApiPropsList>

#### MenubarContent

<ApiPropsList>

<ApiProp
  name="align"
  fullType='"start" | "center" | "end"'
  simpleType="string"
  defaultValue='"start"'
>
  How to align the menu. Flattened from Positioner. Base UI defaults to `'center'`.
</ApiProp>

<ApiProp
  name="alignOffset"
  fullType="number"
  defaultValue="-4"
>
  Offset along alignment axis. Flattened from Positioner.
</ApiProp>

<ApiProp
  name="sideOffset"
  fullType="number"
  defaultValue="8"
>
  Distance in pixels for positioning. Flattened from Positioner. Base UI defaults to `0`.
</ApiProp>

<ApiProp name="level" fullType="1 | 2 | 3 | 4 | 5 | 6 | 7 | 8" defaultValue="3">
  Surface elevation level for the dropdown popup.
</ApiProp>

<ApiProp name="shadowLevel" fullType="1 | 2 | 3 | 4 | 5 | 6 | 7 | 8" defaultValue="3">
  Shadow weight for the dropdown popup.
</ApiProp>

</ApiPropsList>

#### MenubarSubContent

<ApiPropsList>

<ApiProp name="shadowLevel" fullType="1 | 2 | 3 | 4 | 5 | 6 | 7 | 8" defaultValue="3">

</ApiPropsList>

### Modified Defaults

Our implementation changes the following Base UI prop defaults:

- **MenubarTrigger**: `delay` defaults to `0` (Base UI defaults to `600`)
- **MenubarTrigger**: `closeDelay` defaults to `0`
- **MenubarContent**: `align` defaults to `'start'` (Base UI defaults to `'center'`)
- **MenubarContent**: `alignOffset` defaults to `-4`
- **MenubarContent**: `sideOffset` defaults to `8` (Base UI defaults to `0`)
- **MenubarSubContent**: `sideOffset` defaults to `8`
