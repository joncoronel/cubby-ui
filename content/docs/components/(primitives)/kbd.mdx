import { Kbd } from "@/registry/default/kbd/kbd";

Kbd renders a keyboard key or shortcut with key-cap styling. Use it in docs, tooltips, command menus, and shortcut lists to show combinations like `⌘ K` or `Ctrl + S`. Pass `keys` to render a combination, and it converts named keys to platform symbols automatically.

## Examples

### Size Variants

Use `size` to set the key's height and text size.

<ComponentPreview component="kbd" example="kbd-size-variants" />

| ---- | ---------------------------- |

### Color Variants

Use `variant` to set the key's color scheme.

<ComponentPreview component="kbd" example="kbd-color-variants" />

| Variant     | Description                              |
| ----------- | ---------------------------------------- |
| `default`   | Card-colored key with a subtle border.   |
| `primary`   | Solid primary-colored key.               |
| `secondary` | Solid secondary-colored key.             |
| `outline`   | Borderless, transparent key.             |
| `ghost`     | Muted key with no border or shadow.      |
| `danger`    | Destructive-colored key.                 |

## API Reference

Kbd is a custom styled `kbd` element. All props are custom or standard `kbd` attributes passed through.

### Props

#### Kbd

Keyboard key or shortcut with key-cap styling.

<ApiPropsList>

<ApiProp name="size" fullType='"sm" | "md" | "lg"' simpleType="string" defaultValue='"md"'>
  Height and text size of the key.
</ApiProp>

<ApiProp
  name="variant"
  fullType='"default" | "primary" | "secondary" | "outline" | "ghost" | "danger"'
  simpleType="string"
  defaultValue='"default"'
>
  Color scheme of the key.
</ApiProp>

<ApiProp name="pressed" fullType="boolean" defaultValue="false">
  Shows the key in a pressed-down state.
</ApiProp>

<ApiProp name="keys" fullType="string[]" simpleType="array">
  Keys to render as a combination, joined by `separator`. When set, `children` is ignored. Named keys convert to platform symbols.
</ApiProp>

<ApiProp name="separator" fullType="string" defaultValue='"+"'>
  Symbol placed between keys when rendering a `keys` combination.
</ApiProp>

<ApiProp name="platform" fullType='"mac" | "windows" | "auto"' simpleType="string" defaultValue='"auto"'>
  Symbol set used for named keys. `"auto"` detects the user's OS.
</ApiProp>

<ApiProp name="disabled" fullType="boolean" defaultValue="false">
  Dims the key and sets `aria-disabled` to mark it unavailable.
</ApiProp>

<ApiProp name="aria-label" fullType="string" simpleType="string">
  Accessible label for the key, useful when the visible content is a symbol.
</ApiProp>

</ApiPropsList>
