Tabs split related content into panels and show one at a time behind a row of triggers. Use them to organize a settings screen, a card's detail views, or any section where the panels are peers and only one is relevant at once.

## Composition

```text
Tabs
├── TabsList
│   └── TabsTrigger
└── TabsPanels
    └── TabsContent
```

Wrap all `TabsContent` panels in `TabsPanels` to get the shared grid layout and directional slide transitions. `TabsList` renders the active indicator internally.

## Examples

### Sizes

Use `size` on `TabsList` to switch between `small` and `medium`; the list and indicator corner radius scale with it.

<ComponentPreview component="tabs" example="tabs-sizes" />

| Size     | Description                            |
| -------- | -------------------------------------- |
| `small`  | Compact triggers with tighter padding. |
| `medium` | The default, roomier triggers.         |

### Controlled

Use `value` and `onValueChange` on `Tabs` to drive the active tab yourself.

<ComponentPreview component="tabs" example="tabs-controlled" />

### Underline Variant

Set `variant="underline"` on `TabsList` for an underline indicator instead of the capsule.

<ComponentPreview component="tabs" example="tabs-underline-variant" />

| Variant     | Description                                          |
| ----------- | --------------------------------------------------- |
| `capsule`   | The default, a filled pill behind the active tab.   |
| `underline` | A bar under (or beside) the active tab.             |

### Vertical Layout

Set `orientation="vertical"` on `Tabs` to stack triggers down the side.

<ComponentPreview component="tabs" example="tabs-vertical" />

### Vertical Underline

Combine `orientation="vertical"` with `variant="underline"` for a side rail.

<ComponentPreview component="tabs" example="tabs-vertical-underline" />

### Vertical Underline with Sides

Use `side` on `TabsList` to place the underline on the `left` or `right` edge.

<ComponentPreview component="tabs" example="tabs-vertical-underline-with-sides" />

### Animated

`TabsPanels` slides and fades between panels in the direction of the move.

## API Reference

The Tabs component is built on top of [Base UI's Tabs](https://base-ui.com/react/components/tabs). All Base UI props are supported. The documentation below only covers custom props and modified defaults specific to our implementation.

For the complete Base UI API, see the [Base UI Tabs documentation](https://base-ui.com/react/components/tabs).

### Props

#### TabsList

Container for tab triggers with visual styling. Wraps Base UI's `Tabs.List`.

<ApiPropsList>

<ApiProp
  name="variant"
  fullType="'capsule' | 'underline'"
  defaultValue="'capsule'"
>
  Visual style variant for the tabs list. Custom prop not available in Base UI.
</ApiProp>

<ApiProp
  name="size"
  fullType="'small' | 'medium'"
  defaultValue="'medium'"
>
  Size variant for the tabs. Custom prop not available in Base UI.
</ApiProp>

<ApiProp
  name="side"
  fullType="'left' | 'right'"
  defaultValue="'left'"
>
  Side positioning for vertical tab orientation. Custom prop not available in Base UI.
</ApiProp>

</ApiPropsList>

#### TabsPanels

Grid container for tab panels. Wrap all `TabsContent` items in `TabsPanels` to enable proper layout and smooth directional slide and fade transitions between panels.

#### TabsContent

Panel that displays content for the active tab. Wraps Base UI's `Tabs.Panel`.
