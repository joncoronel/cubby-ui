Breadcrumbs show where the current page sits in a site's hierarchy and let users jump back to any ancestor. Use them on deep, nested routes such as docs sections, dashboards, or file paths.

## Accessibility

The `Breadcrumb` nav defaults to `aria-label="breadcrumb"`, and `BreadcrumbPage` sets `aria-current="page"` to mark the active location. Separators are hidden from assistive tech with `aria-hidden`, and `BreadcrumbEllipsis` exposes a screen-reader label ("More pages" by default) for the collapsed items.
