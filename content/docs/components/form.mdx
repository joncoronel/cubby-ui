## API Reference

The Form component is built on top of [Base UI's Form](https://base-ui.com/react/components/form). All Base UI props are supported.

For the complete Base UI API, see the [Base UI Form documentation](https://base-ui.com/react/components/form).

### Props

#### Form

A native form element with consolidated error handling. Renders a `<form>` element.

<ApiPropsList>

<ApiProp name="onFormSubmit" fullType="(formValues: Record<string, any>, eventDetails: SubmitEventDetails) => void">
  Called when the form is submitted. Receives form values as a JavaScript object. `preventDefault()` is automatically called on the native submit event.
</ApiProp>

<ApiProp name="errors" fullType="Record<string, string | string[]>">
  Server-side validation errors. Keys correspond to field `name` attributes. Errors are cleared from a field when its value changes.
</ApiProp>

<ApiProp name="validationMode" fullType="'onSubmit' | 'onBlur' | 'onChange'" defaultValue="'onSubmit'">
  When fields should be validated. Individual fields can override this with their own `validationMode`.
</ApiProp>

</ApiPropsList>
