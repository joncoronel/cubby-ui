## API Reference

The Avatar component is built on top of [Base UI's Avatar](https://base-ui.com/react/components/avatar). All Base UI props are supported. The documentation below only covers custom props specific to our implementation.

For the complete Base UI API, see the [Base UI Avatar documentation](https://base-ui.com/react/components/avatar).

### Custom Props

#### Avatar

<ApiPropsList>

<ApiProp name="size" fullType='"sm" | "md" | "lg"' simpleType="string" defaultValue='"md"'>
