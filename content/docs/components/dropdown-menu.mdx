## API Reference

The DropdownMenu component is built on top of [Base UI's Menu](https://base-ui.com/react/components/menu). All Base UI props are supported. The documentation below only covers custom props and modified defaults specific to our implementation.

For the complete Base UI API, see the [Base UI Menu documentation](https://base-ui.com/react/components/menu).

### Props

#### DropdownMenuContent

Dropdown container with positioning and animations. Composes `Menu.Portal`, `Menu.Positioner`, and `Menu.Popup`. Props are forwarded to `Menu.Popup`.

<ApiPropsList>

<ApiProp
  name="side"
  fullType='"top" | "right" | "bottom" | "left"'
  simpleType="string"
  defaultValue='"bottom"'
>
  Which side of the trigger to position the menu. This prop is flattened from
  the Positioner component for convenience.
</ApiProp>

<ApiProp
  name="align"
  fullType='"start" | "center" | "end"'
  simpleType="string"
  defaultValue='"center"'
>
  How to align the menu relative to the trigger. This prop is flattened from the
  Positioner component for convenience.
</ApiProp>

<ApiProp name="sideOffset" fullType="number" defaultValue="4">
  Distance in pixels between trigger and menu. This prop is flattened from the
  Positioner component for convenience.
</ApiProp>

</ApiPropsList>

#### DropdownMenuItem

Individual menu item. Wraps Base UI's `Menu.Item`.

<ApiPropsList>

<ApiProp name="inset" fullType="boolean">
  Add left padding to align with items that have icons, creating consistent
  visual alignment.
</ApiProp>

<ApiProp
  name="variant"
  fullType='"default" | "destructive"'
  simpleType="string"
  defaultValue='"default"'
>
  Visual style variant. Use `"destructive"` for dangerous actions like delete or
  remove operations.
</ApiProp>

</ApiPropsList>

#### DropdownMenuLinkItem

Menu item that renders as an anchor (`<a>`) for navigation links. Wraps Base UI's `Menu.LinkItem`.

<ApiPropsList>

<ApiProp name="inset" fullType="boolean">
  Add left padding to align with items that have icons, creating consistent
  visual alignment.
</ApiProp>

</ApiPropsList>

#### DropdownMenuLabel

Non-interactive label for grouping items.

<ApiPropsList>

<ApiProp name="inset" fullType="boolean">
  Add left padding to align with items that have icons, creating consistent
  visual alignment.
</ApiProp>

</ApiPropsList>

#### DropdownMenuGroupLabel

Label for a menu group. Wraps Base UI's `Menu.GroupLabel`.

<ApiPropsList>

<ApiProp name="inset" fullType="boolean">
  Add left padding to align with items that have icons, creating consistent
  visual alignment.
</ApiProp>

</ApiPropsList>

#### DropdownMenuSubTrigger

Item that opens a submenu on hover. Wraps Base UI's `Menu.SubmenuTrigger`.

<ApiPropsList>

<ApiProp name="inset" fullType="boolean">
  Add left padding to align with items that have icons, creating consistent
  visual alignment.
</ApiProp>

<ApiProp name="delay" fullType="number" defaultValue="0">
  Delay in milliseconds before opening the submenu on hover.
</ApiProp>

<ApiProp name="closeDelay" fullType="number" defaultValue="0">
  Delay in milliseconds before closing the submenu when hovering away.
</ApiProp>

</ApiPropsList>

#### DropdownMenuSubContent

Submenu container with positioning. Composes `Menu.Portal`, `Menu.Positioner`, and `Menu.Popup`. Props are forwarded to `Menu.Popup`.

<ApiPropsList>

<ApiProp name="sideOffset" fullType="number" defaultValue="0">
  Distance in pixels for positioning the submenu relative to its trigger.
</ApiProp>

<ApiProp
  name="align"
  fullType='"start" | "center" | "end"'
  simpleType="string"
  defaultValue='"start"'
>
  How to align the submenu relative to its trigger.
</ApiProp>

<ApiProp name="alignOffset" fullType="number" defaultValue="-5">
  Vertical offset to align the first submenu item with its trigger. Defaults to
  `-5` when `align` is not `"center"`.
</ApiProp>

</ApiPropsList>
