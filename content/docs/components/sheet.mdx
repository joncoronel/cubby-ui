### Floating Variant

Use `variant="floating"` for a card-like appearance with rounded corners and margins from screen edges. Great for action sheets and share menus.

<ComponentPreview component="sheet" example="sheet-variants" />

### With Form

Include form elements within the sheet.

<ComponentPreview component="sheet" example="sheet-with-form" />

### Nested Sheets

Sheets can be nested within each other. When a nested sheet opens, the parent sheet scales down slightly and shifts away from its edge, revealing a sliver of it behind the nested sheet.

<ComponentPreview component="sheet" example="sheet-nested" />

### Inset Footer

Use `footerVariant="inset"` to give the footer a muted background with a top border, visually separating it from the content.

<ComponentPreview component="sheet" example="sheet-inset-footer" />

## API Reference

The Sheet component is built on top of [Base UI's Dialog](https://base-ui.com/react/components/dialog). All Base UI props are supported. The documentation below only covers custom props specific to our implementation.

For the complete Base UI API, see the [Base UI Dialog documentation](https://base-ui.com/react/components/dialog).

### Custom Props

#### SheetContent

<ApiPropsList>

<ApiProp name="side" fullType='"top" | "right" | "bottom" | "left"' simpleType="string" defaultValue='"right"'>
  Which side of the screen the sheet slides in from. Custom prop not available in Base UI.
</ApiProp>

<ApiProp name="variant" fullType='"default" | "floating"' simpleType="string" defaultValue='"default"'>
  Visual style variant. `"floating"` adds margins from screen edges and rounded corners for a card-like appearance.
</ApiProp>

<ApiProp name="footerVariant" fullType='"default" | "inset"' simpleType="string" defaultValue='"default"'>
  Footer style variant. `"inset"` gives the footer a muted background with a top border for visual separation.
</ApiProp>

<ApiProp name="showCloseButton" fullType="boolean" simpleType="boolean" defaultValue="true">
  Whether to show the close button in the top-right corner.
</ApiProp>

</ApiPropsList>
