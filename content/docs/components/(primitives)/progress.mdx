title: Progress

import { Progress } from "@/registry/default/progress/progress";

A progress bar shows how far along a task or operation is, from a starting value to completion. Use it for uploads, downloads, or any determinate loading; for a static reading within a fixed range, use [`Meter`](/docs/components/meter) instead.

## Examples

### Different Sizes

Use `size` to set the track height and text size.

<ComponentPreview component="progress" example="progress-different-sizes" />

| Value | Description                       |
| ----- | --------------------------------- |
| `sm`  | Thin track with extra-small text. |
| `md`  | The default track and text.       |
| `lg`  | Thick track with larger text.     |

### Custom Colors

Pass a `className` to `ProgressIndicator` to recolor the filled bar.

<ComponentPreview component="progress" example="progress-custom-colors" />

### With Label

Add `ProgressLabel` and `ProgressValue` to caption the bar.

<ComponentPreview component="progress" example="progress-with-label" />

### Dynamic Progress

Set `animated` on `ProgressRoot` to ease the indicator between values as they update.

### File Upload

## API Reference

The Progress component is built on top of [Base UI's Progress](https://base-ui.com/react/components/progress). All Base UI props are supported. The documentation below only covers custom props and modified defaults specific to our implementation.

For the complete Base UI API, see the [Base UI Progress documentation](https://base-ui.com/react/components/progress).

### Props

#### ProgressRoot

<ApiPropsList>

<ApiProp name="animated" fullType="boolean" defaultValue="false">
  Ease the indicator between values as it updates.
</ApiProp>

<ApiProp
  name="size"
  fullType='"sm" | "md" | "lg"'
  simpleType="string"
  defaultValue='"md"'
>
  Size of the progress bar.
</ApiProp>

<ApiProp name="min" fullType="number" defaultValue="0">
  Minimum value of the progress range.
</ApiProp>

<ApiProp name="max" fullType="number" defaultValue="100">
  Maximum value of the progress range.
</ApiProp>

</ApiPropsList>

#### ProgressTrack

<ApiPropsList>

<ApiProp name="striped" fullType="boolean">
  Add a striped pattern to the track.
</ApiProp>

</ApiPropsList>

#### ProgressValue

<ApiPropsList>

<ApiProp
  name="format"
  fullType="(value: number, max: number) => string"
  simpleType="function"
>
  Format the displayed value. Receives the current value and max as arguments.
</ApiProp>

</ApiPropsList>
