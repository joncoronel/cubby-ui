## API Reference

The Tabs component is built on top of [Base UI's Tabs](https://base-ui.com/react/components/tabs). All Base UI props are supported. The documentation below only covers custom props and modified defaults specific to our implementation.

For the complete Base UI API, see the [Base UI Tabs documentation](https://base-ui.com/react/components/tabs).

### Props

#### TabsList

Container for tab triggers with visual styling. Wraps Base UI's `Tabs.List`.

<ApiPropsList>

<ApiProp
  name="variant"
  fullType="'capsule' | 'underline'"
  defaultValue="'capsule'"
>
  Visual style variant for the tabs list. Custom prop not available in Base UI.
</ApiProp>

<ApiProp
  name="size"
  fullType="'small' | 'medium'"
  defaultValue="'medium'"
>
  Size variant for the tabs. Custom prop not available in Base UI.
</ApiProp>

<ApiProp
  name="side"
  fullType="'left' | 'right'"
  defaultValue="'left'"
>
  Side positioning for vertical tab orientation. Custom prop not available in Base UI.
</ApiProp>

</ApiPropsList>

#### TabsContent

Panel that displays content for the active tab. Wraps Base UI's `Tabs.Panel`.

<ApiPropsList>

<ApiProp
  name="animate"
  fullType="boolean"
  defaultValue="false"
>
  Enable animated transitions when switching between tabs. Custom prop not available in Base UI.
</ApiProp>

</ApiPropsList>
