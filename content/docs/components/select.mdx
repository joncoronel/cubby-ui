## API Reference

The Select component is built on top of [Base UI's Select](https://base-ui.com/react/components/select). All Base UI props are supported. The documentation below only covers custom props and modified defaults specific to our implementation.

For the complete Base UI API, see the [Base UI Select documentation](https://base-ui.com/react/components/select).

### Props

#### SelectValue

Displays the selected value or placeholder text. Wraps Base UI's `Select.Value`.

<ApiPropsList>

<ApiProp
  name="items"
  fullType="Record<string, string> | { label: string; value: string }[]"
  simpleType="object | array"
>
  Item mapping for rendering multi-select values. Maps value keys to display
  labels.
</ApiProp>

<ApiProp name="placeholder" fullType="string" simpleType="string">
  Placeholder text displayed when no value is selected.
</ApiProp>

</ApiPropsList>

#### SelectTrigger

Button that opens the dropdown when clicked. Wraps Base UI's `Select.Trigger`.

<ApiPropsList>

<ApiProp
  name="size"
  fullType="'sm' | 'default'"
  simpleType="string"
  defaultValue='"default"'
>
  Size variant of the trigger button.
</ApiProp>

<ApiProp name="hideChevronRotation" fullType="boolean" defaultValue="false">
  Disable the chevron rotation animation when the select opens.
</ApiProp>

</ApiPropsList>

#### SelectContent

Dropdown container for the option list with positioning. Composes Base UI's `Select.Positioner` and `Select.Popup`. Props are forwarded to `Select.Popup`.

<ApiPropsList>

<ApiProp
  name="size"
  fullType="'sm' | 'default'"
  simpleType="string"
>
  Size variant for the popup items. Should match the trigger's size when using
  `alignItemWithTrigger` to ensure proper alignment.
</ApiProp>

<ApiProp name="alignItemWithTrigger" fullType="boolean" defaultValue="false">
  Align the selected item with the trigger button position. Enables native-style
  scrolling with scroll arrows.
</ApiProp>

<ApiProp
  name="fadeEdges"
  fullType="boolean | 'top' | 'bottom' | 'left' | 'right' | 'x' | 'y' | FadeEdge[]"
  defaultValue="true"
>
  Controls the fade effect on scroll edges. See
  [ScrollArea](/docs/components/scroll-area#custom-props) for all options.
</ApiProp>

<ApiProp name="scrollbarGutter" fullType="boolean" defaultValue="false">
  Reserves space for the scrollbar when content overflows, preventing layout
  shift.
</ApiProp>

<ApiProp name="persistScrollbar" fullType="boolean" defaultValue="false">
  Always show the scrollbar instead of fading it in on hover or scroll.
</ApiProp>

<ApiProp name="hideScrollbar" fullType="boolean" defaultValue="false">
  Hides the scrollbar while keeping scroll functionality.
</ApiProp>

<ApiProp name="nativeScroll" fullType="boolean" defaultValue="false">
  Use native browser scrolling with styled scrollbars instead of Base UI's
  scroll virtualization. Enables CSS scroll-driven animations for fade edges.
</ApiProp>

<ApiProp name="sideOffset" fullType="number" defaultValue="6">
  Distance between the trigger and the popup. Base UI defaults to `0`.
</ApiProp>

</ApiPropsList>
