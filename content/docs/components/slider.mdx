### Children Layout

Children render **above** the track, not below. This enables the common pattern of a header row (`SliderLabel` + `SliderValue` side by side) sitting directly above the slider control:

```tsx
<Slider defaultValue={[50]}>
  <div className="flex items-center justify-between">
    <SliderLabel>Volume</SliderLabel>
    <SliderValue />
  </div>
</Slider>
```

If you want content *below* the track, render it as a sibling of the `Slider` instead of a child — for example, a `<FieldDescription>` inside the enclosing `<Field>`.

### With Reference Scale

Render annotations (like a reference scale) as a sibling of `Slider` — content outside the `Slider` root doesn't need the slider's context, so it avoids the label/value composition rules. A `SliderLabel` child keeps the visible label above the track.

### Form Integration

Slider works with two submission flows. With Base UI's `onFormSubmit`, values come from registered [Field](/docs/components/field)s — wrap each slider in `<Field name="...">` (and compose with [Fieldset](/docs/components/fieldset) for multi-thumb groups so each thumb keeps its own `aria-label` under a shared legend). With a native `onSubmit`, Slider's hidden input makes `name` sufficient on its own. The example below shows both. See the [Forms guide](/docs/getting-started/forms) for more patterns.

## API Reference

### Slider

<ApiPropsList>

<ApiProp
  name="variant"
  fullType='"default" | "contained" | "squareThumb"'
  simpleType="string"
  defaultValue='"default"'
>
  Visual style variant for the slider. Choose from default, contained (thumb
  aligned to edges), or squareThumb styles.
</ApiProp>

<ApiProp name="getAriaLabel" fullType="((index: number) => string) | null">
  A function that returns an accessible name for each thumb. Use for sliders without a visible label, or for range sliders where each thumb needs a distinct label.
</ApiProp>

<ApiProp name="showSteps" fullType="boolean" defaultValue="false">
  Show step indicator dots along the slider track for discrete value
  visualization.
</ApiProp>

<ApiProp name="min" fullType="number" defaultValue="0">
  Minimum value of the slider range.
</ApiProp>

<ApiProp name="max" fullType="number" defaultValue="100">
  Maximum value of the slider range.
</ApiProp>

<ApiProp name="thumbAlignment" fullType='"center" | "edge"' simpleType="string">
  Alignment of the slider thumb. Automatically set to 'edge' when variant is
  'contained', otherwise defaults to 'center'.
</ApiProp>

</ApiPropsList>

### Additional Props

Slider inherits all props from [Base UI Slider](https://base-ui.com/react/components/slider#api-reference), including orientation, step, value, defaultValue, onChange, and accessibility props.
