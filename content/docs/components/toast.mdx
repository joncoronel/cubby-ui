## API Reference

The Toast component is built on top of [Base UI's Toast](https://base-ui.com/components/toast). All Base UI props are supported. The documentation below only covers custom props and modified defaults specific to our implementation.

For the complete Base UI API, see the [Base UI Toast documentation](https://base-ui.com/components/toast).

### toast() Function

The primary API is the `toast()` function, which programmatically displays toast notifications. It supports two call signatures:

**1. Object-based (recommended)**

```tsx
toast({
  title: "Success",
  description: "Your changes have been saved",
  type: "success",
});
```

**2. JSX-based**

```tsx
toast(<div>Custom JSX content</div>, { type: "info" });
```

### ToastOptions

| Prop          | Type                                                       | Default     | Description                                               |
| ------------- | ---------------------------------------------------------- | ----------- | --------------------------------------------------------- |
| `title`       | `string`                                                   | -           | Toast title                                               |
| `description` | `string`                                                   | -           | Toast description                                         |
| `type`        | `"default" \| "success" \| "error" \| "warning" \| "info"` | `"default"` | Visual type of the toast                                  |
| `duration`    | `number`                                                   | `5000`      | Duration in milliseconds before auto-dismissal            |
| `priority`    | `"low" \| "high"`                                          | `"low"`     | Display priority (high priority toasts appear at the top) |
| `action`      | `{ label: string; onClick: () => void }`                   | -           | Action button configuration                               |
| `data`        | `TData`                                                    | -           | Custom data attached to the toast                         |
| `onClose`     | `() => void`                                               | -           | Callback when toast closes                                |
| `onRemove`    | `() => void`                                               | -           | Callback when toast is removed from DOM                   |

### Helper Methods

Type-specific helper methods for common toast types:

```tsx
// Type variants
toast.success({ title: "Saved!" });
toast.error({ title: "Failed" });
toast.warning({ title: "Warning" });
toast.info({ title: "Info" });

// Promise handling (simple)
toast.promise(fetchData(), {
  loading: "Loading...",
  success: "Data loaded!",
  error: "Failed to load",
});

// Promise handling (with title and description)
toast.promise(fetchData(), {
  loading: { title: "Loading...", description: "Fetching data" },
  success: (data) => ({ title: "Success!", description: `Loaded ${data.name}` }),
  error: (err) => ({ title: "Error", description: err.message }),
});

// Programmatic control
toast.dismiss(toastId);
toast.update(toastId, { title: "Updated" });
```

### Anchored Toasts

Show toasts anchored to a specific element:

```tsx
toast.anchored({
  description: "Copied!",
  anchor: buttonRef,
  side: "top",
  sideOffset: 8,
  arrow: true,
  duration: 2000,
});
```

| Prop          | Type                                     | Default    | Description                      |
| ------------- | ---------------------------------------- | ---------- | -------------------------------- |
| `anchor`      | `Element \| RefObject<Element>`          | -          | Element to anchor the toast to   |
| `side`        | `"top" \| "bottom" \| "left" \| "right"` | `"top"`    | Side of anchor to position toast |
| `sideOffset`  | `number`                                 | `8`        | Offset from anchor               |
| `align`       | `"start" \| "center" \| "end"`           | `"center"` | Alignment along the anchor       |
| `alignOffset` | `number`                                 | `0`        | Offset for alignment             |
| `arrow`       | `boolean`                                | `false`    | Show arrow pointing to anchor    |

### ToastProvider

Wraps your app to enable stacked toast notifications.

| Prop        | Type                                                                                              | Default          | Description                        |
| ----------- | ------------------------------------------------------------------------------------------------- | ---------------- | ---------------------------------- |
| `position`  | `"top-left" \| "top-center" \| "top-right" \| "bottom-left" \| "bottom-center" \| "bottom-right"` | `"bottom-right"` | Position of the toast viewport     |
| `limit`     | `number`                                                                                          | `3`              | Maximum number of toasts visible   |
| `timeout`   | `number`                                                                                          | `5000`           | Default timeout for toasts in ms   |
| `container` | `HTMLElement \| RefObject<HTMLElement>`                                                           | -                | Custom portal container            |

### AnchoredToastProvider

Wraps your app to enable anchored toasts (positioned relative to elements).

| Prop        | Type                                    | Default | Description             |
| ----------- | --------------------------------------- | ------- | ----------------------- |
| `timeout`   | `number`                                | `5000`  | Default timeout in ms   |
| `container` | `HTMLElement \| RefObject<HTMLElement>` | -       | Custom portal container |

### Setup

Wrap your app with both providers to enable all toast functionality:

```tsx
// app/layout.tsx or similar
import { ToastProvider, AnchoredToastProvider } from "@/registry/default/toast/toast";

export default function RootLayout({ children }) {
  return (
    <html>
      <body>
        <ToastProvider position="bottom-right">
          <AnchoredToastProvider>
            {children}
          </AnchoredToastProvider>
        </ToastProvider>
      </body>
    </html>
  );
}
```
